# Subodh | Personal Portfolio Website

A modern, fully responsive personal portfolio website built using HTML, CSS, and JavaScript.  
The website features a premium dark theme, structured layout, smooth animations, glass-style navbar, and interactive UI effects.

Live Website: https://your-vercel-link-here  
GitHub Repository: https://github.com/Subodh9999/Portfolio-website

---

## Overview

This portfolio website is designed to showcase freelance web development projects with a clean user experience and optimized performance.  
It includes modern UI elements, responsive design principles, and conversion-focused call-to-action sections.

---

## Features

- Modern dark theme design
- Gradient and structured background effects
- Glassmorphism-style navbar with blur
- Apple-style hide and reveal navbar on scroll
- Smooth scrolling and reveal animations
- Typing animation in hero section
- Responsive layout for all screen sizes
- Interactive project cards with hover effects
- Contact form integration
- WhatsApp and Email call-to-action buttons
- Optimized performance and SEO structure

---

## Tech Stack

- HTML5
- CSS3 (Flexbox and Responsive Design)
- JavaScript (Vanilla JS)
- FormSubmit (for contact form handling)
- Vercel (Deployment)
